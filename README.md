# Buscador de Imágenes con Formik y la API de Unsplash

Buscador de imagenes utilizando formik y la api de Unsplash para la obtencion de imagenes de alta calidad

### Tecnoloías utilizadas
- React
- Formik
- API de Unsplash

### Dependencias
` npm install formik `
